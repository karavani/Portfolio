# 🌍 זיהוי שפה אוטומטי

## מה השתנה?

### ✅ תיקונים שבוצעו:

1. **תוכן נעלם באנגלית** - תוקן!
   - `Home.js` היה משתמש בקובץ תרגומים ישן וחלקי
   - עכשיו כל הקומפוננטות משתמשות ב-`translations/translations.js`
   - נמחק הקובץ `translations/index.js` הישן

2. **זיהוי שפה אוטומטי** - נוסף!
   - המערכת מזהה אוטומטית את שפת הדפדפן של המשתמש
   - שומרת את העדפת השפה ב-`localStorage`
   - תומכת בזיהוי עברית (he/iw)

---

## 🔧 איך זה עובד?

### זרימת זיהוי השפה:

1. **בדיקה ראשונה**: האם יש שפה שמורה ב-localStorage?
   - אם כן ← משתמש בה
   - אם לא ← ממשיך לשלב 2

2. **בדיקה שנייה**: מה שפת הדפדפן?
   - עברית (he/iw) ← מגדיר עברית
   - אחר ← מגדיר אנגלית

3. **שמירה**: כל שינוי שפה נשמר ב-localStorage

---

## 🎯 דוגמאות שימוש

### דוגמה 1: משתמש ישראלי חדש
```
1. נכנס לאתר לראשונה
2. הדפדפן שלו מוגדר לעברית → האתר בעברית
3. העדפה נשמרת ב-localStorage
4. בכניסה הבאה → עדיין בעברית
```

### דוגמה 2: משתמש שמחליף שפה
```
1. לוחץ על כפתור החלפת שפה
2. השפה משתנה → נשמר ב-localStorage
3. ריענון הדף → השפה נשארת כמו שבחר
```

### דוגמה 3: משתמש זר
```
1. נכנס לאתר עם דפדפן באנגלית
2. האתר באנגלית
3. יכול להחליף לעברית אם רוצה
```
